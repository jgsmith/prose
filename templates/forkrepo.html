Fork the Repository

For DHData.Org to accept your contributions, you need to go through the following steps. Don't worry. There is no software to download or install on your computer.

Step 1: Fork the "dhdata-site" repository

To fork the repository for this site, go to the GitHub page for the repository and click the "Fork" button. This will create your own copy of the site.

Step 2: Return to this editor

After forking the site repository, return to this editor. It will detect that you have forked the repository and will show you the different types of content that you may contribute or edit.

Step 3: Write your content

Now you may create new recipes, datasets, or other types of content. If you are creating new content, the editor will provide you with an appropriate template, so don't worry about having to remember all of the different parts.

Step 4: Create a pull request

After you have written or edited the content you would like to contribute, you will need to create a pull request using your fork in your GitHub account. Feel free to read through the GitHub help page for creating a pull request.
